## Ce qui est livré (mai 2026)

### Scraper
- **La Bonne Alternance** (`api.apprentissage.beta.gouv.fr`) — ROME M1802/M1805/M1806/M1810
- **France Travail API v2** (`api.francetravail.io`) — OAuth2 client_credentials
- Extraction `description` + `desired_skills` pour les deux sources
- Tags extraits automatiquement depuis titre + description + compétences
- Upsert Supabase via `requests` (HTTP/1.1), batches de 25, cron toutes les 6h
- ~200 offres actives avec description enrichie

### Frontend
- Grille d'offres avec filtres : localisation, tags, date, durée
- **Modal offre** : description complète, compétences, bouton "Postuler"
- **Score de matching** : badge % sur chaque carte (vert ≥70%, amber ≥40%)
- **Forces & gaps** : dans le modal, mots-clés qui matchent vs ce qui manque
- Export CSV des offres filtrées

### Matching CV × Offres
- **CLI Python** : `python tools/run_matching.py --cv mon_cv.pdf`
- **Frontend** : upload CV sur `/suivi`, scores persistés en `localStorage`
- TF-IDF cosine similarity, stop words FR, titre × 3, scores normalisés 0–100%
- Insights par offre : forces (termes communs) + gaps (termes manquants)

### Kanban `/suivi`
- 6 colonnes : Intéressé → Candidaté → Relancé → Entretien → Accepté → Refusé
- Drag-and-drop (dnd-kit), notes, date de relance
- Score de matching affiché sur chaque carte
- Auth Supabase — kanban privé par utilisateur (RLS)

### Infrastructure
- Migrations SQL versionnées (`001` → `004`)
- CI GitHub Actions : lint, tests, SAST (Bandit), Gitleaks, Trivy
- Déploiement Vercel automatique sur `main`
